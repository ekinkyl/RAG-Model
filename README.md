# RAG-Model

The Log-Based Q&A System is an advanced application designed to analyze web traffic logs and provide accurate, actionable answers to user queries. By leveraging artificial intelligence and modern web frameworks, the system helps monitor site performance and troubleshoot issues efficiently.

Features

AI-Powered Q&A System: Uses the T5 Transformer Model for natural language understanding and response generation.

Log Analysis: Efficiently parses and analyzes large-scale web traffic logs to retrieve relevant data.

Retrieval-Augmented Generation (RAG): Combines data retrieval and generative AI for improved accuracy and contextual understanding.

High Accuracy: Achieved 83% accuracy during testing, ensuring reliable performance.

Fast and Scalable: Incorporates FAISS for fast similarity searches in large datasets.
